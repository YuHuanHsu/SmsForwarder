# SmsForwarder - 短信轉發器

Android 手機簡訊轉發應用程式。

## 功能說明

監控 Android 手機簡訊、來電、APP 通知，並根據指定規則轉發到其他平台：
- 釘釘群機器人
- 企業微信群機器人  
- 飛書群機器人
- 電子郵件
- Bark
- Webhook
- Telegram 機器人
- Server醬
- PushPlus
- 手機簡訊

支援主動控制功能：遠端發送簡訊、查詢簡訊、查詢通話記錄、查詢聯絡人、查詢電量等。

支援自動任務功能，可根據條件自動執行指定動作。

## 注意事項

本應用程式僅供學習研究使用，請根據當地法律法規合理使用。

SmsForwarder 不會收集任何隱私數據，所有簡訊轉發均在您的設備上進行處理。
